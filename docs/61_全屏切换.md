# 全屏切换

受工业复杂的现场环境影响，工控机可能存在无物理键盘的情况，难以退出/进入全屏。CMS提供全屏切换组件，点击即可将系统全屏/退出全屏。

1）编辑：从组件箱-控制器中拖拽[全屏切换]按钮

2）运行：点击即可将系统全屏/退出全屏
