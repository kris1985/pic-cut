# 关机按钮

工业现场操作工在关闭软件后习惯直接关闭电源，容易造成设备故障，CMS提供关机按钮，支持关闭软件的同时将工控机关机，便捷用户操作

1）编辑：从组件箱-控制器中拖拽[关机]按钮

2）运行：点击按钮后，在不同应用形态上将存在以下差异

  * 本地桌面端，弹窗提示“是否关闭软件并关机”，确认后，完全关闭软件后，关闭操作系统；

  * web端，不支持关机；
