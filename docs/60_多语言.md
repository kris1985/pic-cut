# 多语言

页面查看者可能语言各异，无需为每类语言者配置一个页面，可通过【多语言管理】对页面文本元素进行翻译，即可获得多语言页面，极大复用页面。

1）编辑模式下可通过工具栏-多语言，进行多语言调试预览

2）运行模式下可通过组态语言切换组件，进行语言切换

具体配置使用详见[【多语言管理】](/cms/tutorial/多语言/多语言概述)
