# 指令

## CMS提供的系统内置方法指令​

1）变量

  * 读取变量：获取变量值
  * 写入变量：设置变量
  * 查询变量历史数据：查询变量的指定时刻或指定时间范围的历史记录



2）数据管理

  * 查询数据：查询数据表指定条件的记录（条件为sql中where内容,数据表为数据管理中的表名称非表id
  * 插入数据：插入数据表一条记录
  * 清除数据：清楚数据表



​ 注：条件为sql中where内容,数据表为数据管理中的表名称非表id
