🏃‍♂️ 鞋子图片智能裁剪工具 - 快速使用指南

===============================================
🎯 一键制作EXE可执行程序
===============================================

📋 准备工作:
1. 确保电脑已安装 Python 3.8 或更高版本
2. 下载本项目的所有文件到一个文件夹中

⚡ 快速制作EXE:
1. 双击运行 build.bat 文件
2. 等待自动安装依赖和打包（需要网络）
3. 完成后在 dist 文件夹中找到 "鞋子图片智能裁剪工具.exe"
4. 可以将这个exe文件分享给其他人使用

===============================================
🖥️ 图形界面使用说明
===============================================

🚀 启动程序:
- 双击 "鞋子图片智能裁剪工具.exe"
- 或者运行: python shoe_cropper_gui.py

📁 选择文件夹:
1. 点击 "输入目录" 旁的 "浏览" 按钮
2. 选择包含鞋子图片的文件夹
3. 点击 "输出目录" 旁的 "浏览" 按钮  
4. 选择或创建保存处理后图片的文件夹

⚙️ 设置参数:
- 裁剪比例: 推荐选择"自动选择"
- 图片质量: 推荐选择"高质量"
- 高分辨率模式: 大图片(>2000px)建议勾选

▶️ 开始处理:
1. 点击 "开始裁剪" 按钮
2. 查看进度条和日志信息
3. 完成后点击 "打开输出目录" 查看结果

===============================================
📸 支持的图片格式
===============================================

✅ 输入格式: JPG, JPEG, PNG, BMP, TIFF, WebP
✅ 输出格式: JPEG (高质量)

===============================================
🎨 功能特点
===============================================

🤖 智能检测:
- 自动识别鞋子位置和边界
- 支持各种背景色(白色、灰色、彩色)
- 适应不同颜色的鞋子

🎯 精确裁剪:
- 自动居中，确保鞋子在图片中心
- 精确的4:3或3:4比例控制
- 保持高分辨率和图片质量

⚡ 批量处理:
- 一次处理整个文件夹的图片
- 多线程处理，速度快
- 实时显示进度和结果统计

===============================================
🔧 故障排除
===============================================

❓ 常见问题:

Q: 程序无法启动？
A: 检查是否安装了Python，运行 python --version 确认

Q: 打包失败？  
A: 确保网络连接正常，pip可以下载包

Q: 处理效果不理想？
A: 检查原图背景是否清晰，尝试不同的比例设置

Q: 处理速度慢？
A: 关闭高分辨率模式，选择普通质量

Q: 找不到图片？
A: 检查文件格式是否支持，确保是jpg/png等格式
